## 4. 教育政策口径（已澄清）

- “买市内六区房子”通常更稳，但不等于“所有重点都能报”。
- 普通高中招生口径是三类：
  - 面向市内六区
  - 面向本区
  - 面向全市
- 关键结论：能否报某校，最终看该校当年招生计划范围 + 学籍户籍路径 + 分数位次。
- 海教园属于政策性特殊通道（常见需双籍等条件），不能简单当作普遍规则。
